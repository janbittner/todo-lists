## Description

My first [Nest](https://github.com/nestjs/nest) project - ToDo list REST API.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The app will run on http://localhost:3000. API documentation can be accessed on http://localhost:3000/api#/

No need to run any migrations, just install and run.
